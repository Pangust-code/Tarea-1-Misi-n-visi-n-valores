# Tarea 1 - Mision, Vision y Valores

Este repositorio contiene la entrega de la actividad de **Analisis y Diseno de Sistemas** para la empresa creada **Pangust Systems**.

## Datos academicos

- Asignatura: Analisis y Diseno de Sistemas
- Grupo: 2 - Computacion - CUE
- Integrantes: Daniel Guanga, Leo Vasconez, Xavier Siguachi

## Descripcion breve

Se desarrollo un informe completo de mision, vision y valores para Pangust Systems, incluyendo:

- Construccion de la mision con 9 componentes
- Definicion de 3 valores organizacionales
- Vision a 5 anos
- Tabla de validacion tipo Likert
- Conclusion y bibliografia

Ademas, se preparo un guion explicativo dividido en 3 partes para una presentacion en video de 3 a 5 minutos.
